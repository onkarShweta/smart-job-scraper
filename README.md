# Smart Job Scraper

A simple job scraping and tracking application built with Python and Flask.

## Features
- Scrapes job listings from websites
- Stores data in SQLite database
- REST API for job data
- Search jobs by company
- Web interface for job listings
- Docker container support

## Tech Stack
- Python
- Flask
- SQLite
- BeautifulSoup
- Docker
- HTML/CSS (Bootstrap)

## Run Locally

Clone the repository:

git clone https://github.com/onkarShweta/smart-job-scraper.git

Install dependencies:

pip install -r requirements.txt

Run the app:

python app.py

Open in browser:

http://127.0.0.1:5000
